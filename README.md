Multithreaded work server for Bagpipes. 

Made with: Tokio, Polodb + Sled, Subxt and actix-web  

*use the make file*   

## Run tests:
```shell
$ make test 
```
## Run checks: 
```shell
$ make check 
```


## Format code:
```shell
$ make fmt 
```


## Build repo:
```shell
$ make build 
```


## Check dependencies:  
```shell
$ make checkdep 
```
